# QlikVisualizationProvider

<Description of={QlikVisualizationProvider} />

### Props

<ArgsTable of={QlikVisualizationProvider} />

The `QlikVisualizationProvider` exposes the following properties that can be used with the `useQlikVisualizationContext` hook:

```tsx
const {
    visualizationId,
    setVisualizationId,
    onVisualizationFullscreen,
    setOnVisualizationFullscreen,
    onVisualizationExport,
    setOnVisualizationExport
} = useQlikVisualizationContext()
```
